🏥 Setting Up a Case Study Scenario

1

Enter Title

Give the scenario a clear, descriptive name (e.g., "Case Study 12: Diabetic Ketoacidosis").

2

Define Exhibits (Tabs)

The Exhibits field creates the tabs (History, Vitals, Labs, etc.) that students see on the left side.

Copy & Paste this Template:

Tip: You can rename the keys (e.g., change "History" to "Background"). The key becomes the tab name.
3

Add Linked Questions

Scroll down to the "Questions" section.

  • Click "+ Add another Question" to create a new item.
  • This opens the inline form (blue border).
  • Follow the guide tabs above (MCQ, Drag & Drop, etc.) for specific question types.
4

🆕 Set Question Sequence (NEW!)

For Unfolding Case Studies, NCLEX requires 6 questions delivered in order. Use the new "Scenario Progression" section:

  • Question Number in Scenario: Enter 1-6 to control delivery order
  • Clinical Judgment Function: Select which NCJMM function this tests:
    • Recognize Cues (Question 1)
    • Analyze Cues (Question 2)
    • Prioritize Hypotheses (Question 3)
    • Generate Solutions (Question 4)
    • Take Actions (Question 5)
    • Evaluate Outcomes (Question 6)
⚠️ Important: Questions will be delivered in numerical order (1→2→3→4→5→6) regardless of when you create them!
5

🆕 Add Dynamic Exhibit Updates (NEW!)

Real NCLEX scenarios "unfold" - new information appears as students progress. Use "Exhibit Updates (JSON)" to add or update tabs for specific questions.

Example: Question 2 shows new lab results

{
  "Labs_1200": "Glucose: 180 mg/dL (↓ from 340), pH: 7.32 (↑ from 7.22)"
}

Example: Question 4 adds new provider orders

{
  "Provider_Orders": "1. NS 1L bolus IV STAT\n2. Regular insulin 0.1 units/kg/hr\n3. Continuous cardiac monitoring"
}

Example: Question 6 shows updated vitals

{
  "Vitals_1400": "BP: 142/84, HR: 88, RR: 20, Temp: 98.8°F, O2 Sat: 97%"
}
💡 How it works:
• Base exhibits (from Step 2) show with ALL questions
• Exhibit updates MERGE with base exhibits
• New tabs appear with "NEW" badge and pulsing animation
• Leave empty {} if no updates needed for that question
6

🎯 Complete Sample Scenario

Here's a complete example showing all 6 questions with exhibit updates:

Q# NCJMM Function Exhibit Updates
1 Recognize Cues {} (uses base exhibits)
2 Analyze Cues {"Labs_0800": "..."}
3 Prioritize Hypotheses {} (uses previous labs)
4 Generate Solutions {"Provider_Orders": "..."}
5 Take Actions {} (uses previous orders)
6 Evaluate Outcomes {"Vitals_1400": "...", "Labs_1400": "..."}
✅ Student Experience:
• Welcome modal appears on Question 1
• Progress indicator shows "Question X of 6"
• New tabs appear with "NEW" badge and animation
• All 6 questions delivered consecutively
• After Q6, exam returns to adaptive selection

Multiple Choice (MCQ) & Select All That Apply (SATA)

1

Select Question Type

Choose Multiple Choice Question (MCQ) or Select All That Apply (SATA)

2

Enter Question Text

Place question here:

A client with diabetes has a blood glucose of 45 mg/dL. 
What should the nurse do first?
3

Add Options (JSON Format)

Pre-filled with 4 options (A-D). Just fill in the "text" values:

💡 To add more options (E, F, etc.): Add comma after last item, press Enter, then add:

  {"id": "E", "text": "Your option text here"}
4

Set Correct Answers

For MCQ (one answer):

["B"]

For SATA (multiple answers):

["A", "B", "C"]

🎯 Drag & Drop Rationale Questions

1

Select Question Type

Choose Drag & Drop Rationale (DRAG_DROP_RATIONALE)

2

Enter Question Text

Place question here:

A client with Type 1 Diabetes presents with fruity breath, 
rapid breathing, and blood glucose of 450 mg/dL.

Complete the sentence below.
3

Add Options (Template + Wells)

Drag-drop uses template with placeholders ([well1], [well2], etc.)

💡 Tips:

  • Use [well1], [well2], [well3] as placeholders in the template
  • Each well must have a list of answer choices
  • All options from all wells show in a single list (NCLEX format)
  • Students can drag any option to any well
4

Set Correct Answers (Map Wells)

Specify which option goes in which well:

{
  "well1": "Diabetic Ketoacidosis",
  "well2": "Insulin deficiency and fat metabolism"
}

⚠️ Important: The values must exactly match the text in your answer_wells!

▼ Dropdown Rationale Questions

1

Select Question Type

Choose Dropdown Rationale (DROPDOWN_RATIONALE)

2

Enter Question Text

Place question here:

A client is admitted with heart failure.
Complete the sentence using the dropdown options.
3

Add Options (Template + Dropdowns)

Use template with placeholders ([dropdown1], [dropdown2]) and define options for each.

{
  "template": "The client is experiencing [dropdown1] as evidenced by [dropdown2].",
  "dropdowns": {
    "dropdown1": [
      "Fluid Overload",
      "Dehydration",
      "Anemia"
    ],
    "dropdown2": [
      "Jugular Venous Distention",
      "Dry Mucous Membranes",
      "Pale Skin"
    ]
  }
}

💡 Tips:

  • Use [dropdown1], [dropdown2] as placeholders in the template
  • Define the choices for each dropdown in the dropdowns object
4

Set Correct Answers

Map each dropdown ID to its correct value:

{
  "dropdown1": "Fluid Overload",
  "dropdown2": "Jugular Venous Distention"
}

⚠️ Important: The values must exactly match the text in your options!

田 Matrix Multiple Response Questions

1

Select Question Type

Choose Matrix Multiple Choice (MATRIX_MULTIPLE)

2

Enter Question Text

Place question here:

For each client finding below, click to specify whether the finding is consistent with DKA, UTI, or HHS.
3

Add Options (Rows & Columns)

Define the rows (Findings) and columns (Conditions).

{
  "rows": [
    "Fever",
    "Confusion",
    "Urinary Frequency"
  ],
  "columns": [
    "Diabetic Ketoacidosis",
    "Urinary Tract Infection",
    "Hyperosmolar Hyperglycemic Syndrome"
  ]
}

💡 Info: Only include the text for the rows and columns. The grid will be generated automatically.

4

Set Correct Answers

Specify valid finding (row) and diagnosis (column) pairs:

[
  { "finding": "Fever", "diagnosis": "Urinary Tract Infection" },
  { "finding": "Fever", "diagnosis": "Hyperosmolar Hyperglycemic Syndrome" },
  { "finding": "Confusion", "diagnosis": "Diabetic Ketoacidosis" },
  { "finding": "Confusion", "diagnosis": "Urinary Tract Infection" },
  { "finding": "Confusion", "diagnosis": "Hyperosmolar Hyperglycemic Syndrome" },
  { "finding": "Urinary Frequency", "diagnosis": "Diabetic Ketoacidosis" },
  { "finding": "Urinary Frequency", "diagnosis": "Urinary Tract Infection" },
  { "finding": "Urinary Frequency", "diagnosis": "Hyperosmolar Hyperglycemic Syndrome" }
]

⚠️ Note: List every correct checkmark as a separate object.

5

Add Rationale

Rationale: Explain the connections (e.g., "Fever is classic for infection (UTI) but not typical for DKA...").

🖍️ Highlight Text Questions

1

Select Question Type

Choose Highlight Text (HIGHLIGHT_TEXT)

2

Enter Question Text

Place question here:

Read the findings below. Click to highlight the findings that require immediate intervention.
3

Add Options (Formatted Text)

Wrap EVERY sentence or phrase (both correct and incorrect) in <span id="x" class="highlightable">...</span> tags.

For Example:

{
  "formatted_text": "<p>10:00 AM: Client admitted with abdominal pain. Temperature 101.2°F (38.4°C). Heart rate 110 bpm. Blood pressure 124/76 mmHg. Abdomen rigid and board-like on palpation. Client reports pain is 8/10. Bowel sounds hypoactive in all quadrants.</p>"
}
Tip: Notice how even normal findings like "Client admitted..." (s1) are wrapped so they are clickable distractors!
4

Set Correct Answers

List the IDs of the correct spans ONLY:

["h2"]

⚠️ Note:
• h2 (Low BP) is Correct.
• h1 and h3 are Distractors (Clickable but Wrong).

⚙️ General Settings (All Question Types)

1

Add Rationale (Optional)

Explain why the answer is correct. Example:

DKA occurs when lack of insulin forces fat metabolism, 
creating ketones (fruity breath) and acidosis.
2

Set Difficulty

Range: -3.0 (very easy) to +3.0 (very hard)

  • -2.0 to -1.0: Easy
  • -0.5 to 0.5: Medium
  • 1.0 to 2.0: Hard
3

Select Categories

Choose relevant NCLEX categories (e.g., Pharmacology, Management of Care)

4

Save Your Question

Click "Save and add another" or "Save" at the bottom
